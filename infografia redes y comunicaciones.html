Historia evolución

  • 1960s Conmutación de paquetes (primera idea)
  • 1969 ARPANET — 4 nodos iniciales (UCLA, Stanford, etc.)
  • 1970-80 Protocolos y LAN; Ethernet, TCP/IP
  • 1990s Explosión Web (HTTP) y popularización masiva
  • Hoy SDN, IoT, 5G, fibra óptica masiva
🔹 ARPANET 🔹 TCP/IP 🔹 WWW 🔹 5G

Protocolos

  • IP — «sobre» con direcciones origen/destino
  • TCP — ordena y verifica entrega confiable
  • UDP — rápido, sin conexión (streaming)
  • HTTP/HTTPS — idioma de la web
  • DNS — traduce google.com → IP
  • DHCP — asigna IP automática
📌 DNS: la agenda telefónica

IPv4

32 bits · ej. 192.168.1.1

Estructura: RED . SUBNET . HOST

Clase A 10.0.0.0 (grandes redes)
Clase B 172.16.0.0 (medianas)
Clase C 192.168.1.0 (pequeñas)
Máscara 255.255.255.0 = /24
La máscara define red vs host

📡 Capas en acción

Aplicación (HTTP) ⬇ Transporte (TCP) ⬇ Red (IP) ⬇ Enlace/Física

TCP segmenta, IP direcciona, y el medio físico transporta bits.

💡 Dato curioso: La dirección 127.0.0.1 es "localhost" — tu propio equipo.
🧪 IPv6 nació por falta de direcciones IPv4.
